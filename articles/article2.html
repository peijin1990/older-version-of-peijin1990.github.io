关于class类名的js函数的思考

在慕课网学习时，老师教了一个用js获取class类名的方法，后来自己买书看到了《JavaScript DOM编程艺术》，在这本书里，同样讲解了一种用js获取class名称的方法，下面总结分享一下。

               		function getByClass(parent,clsName){ 
                        //用来存取所有获取到的class为box的元素 
                        var boxArr=new Array(); oElements=parent.getElementsByTagName("*"); 
                        for(var i=0;i<oElements.length;i++){ 
                        if(oElements[i].className==clsName){ 
                            boxArr.push(oElements[i]); 
                          } 
                        } 
                         return boxArr; 
                    }

改进注意：

1.这种方法只适用于class类名只有一个的情况，当一个标签含有多个类名时，这种情况会忽略含多个类名的标签。

改为：

                  if(oElements[i].className.indexOf(claName)!=-1){
                    //前面使用了push方法，添加元素到数组中，这里使用直接赋值的方法
                      boxArr[boxArr.length]=oElements[i];
                 }

2.不过这里的函数还不是很完美，因为它忽略了浏览器支持getclassname的情况，理论上，支持的时候，我们其实也可以用不支持的方法获得类名。不过这里我们还是要完善一下，有一个完整的罗辑！

                    function getByClass(node,classname){
                        if(node.getElementsByClassName){
                            return node.getElementsByClassName(classname);
                        }
                        else{
                            var boxArr=[];
                            var elements=node.getElementsByTagName("*");
                            var l=elements.length;
                            for(var i=0;i<l;i++){
                                if(elements[i].className.indexOf(classname)!=-1){
                                    boxArr.push(elements[i]);
                                }
                            }
                            return boxArr;
                        }
                    }
